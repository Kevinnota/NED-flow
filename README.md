# NED-flow

NED-flow is a pipeline developed for using **<u>N</u>**uclear reference genomes to classify reads obtained from shotgun-sequenced ancient **<u>E</u>**nvironmental **<u>D</u>**NA using a Next**<u>flow</u>** backbone. The tool contains tools for database management and building and for preprocessing reads, mapping and taxonomic classifying.
